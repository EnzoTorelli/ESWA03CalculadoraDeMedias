# ESWA12CalculadoraDeMedias

Aplicativo desktop Windows para cálculo de médias semestral e final de alunos.

## Tecnologias
- C# / .NET 8.0
- Windows Forms
- xUnit (testes unitários)
- NuGet

## Projetos
- **ESWA12CalculadoraDeMedias** — Biblioteca de Classes (lógica e validação)
- **CalculadoraDeMedias12** — Aplicativo Windows Forms
- **CalculadoraDeMedias12.Tests** — Testes Unitários xUnit

## Fórmulas
- Média Semestral: `MS = (4×NP1 + 4×NP2 + 2×PIM) / 10`
- Média Final: `MF = (MS + EX) / 2`
